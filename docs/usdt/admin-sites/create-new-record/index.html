1.1.1 添加新记录

指导销售人员为用户创建 USDT 付款记录

步骤 1: 登录管理系统并选择相应选项卡

步骤 2: 点击[请求付款]按钮

步骤 3: 输入信息并提交 t

记录创建后，将显示在用户页面上。
